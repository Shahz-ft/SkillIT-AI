# 🤖 Skill'IT AI

Skill'IT AI is a full-stack AI-powered learning assistant developed to help students study smarter. It provides AI-powered tools such as Ask AI, Notes Summarizer, Quiz Generator, and Flashcards to enhance the learning experience.

🌐 Live Demo: https://skill-it-ai.vercel.app/

---

## 📖 Overview

Skill'IT AI was developed as an extension of **Skill'IT**, an online learning platform originally designed in Figma. The goal was to transform the UI/UX concept into a fully functional AI-powered web application.

The application allows students to:

- 🤖 Ask study-related questions
- 📄 Summarize lengthy notes
- 📝 Generate quizzes from any topic
- 🧠 Create revision flashcards

---

## ✨ Features

### 🤖 Ask AI
Get instant AI-powered explanations for study-related questions.

### 📄 Notes Summarizer
Convert lengthy notes into short and easy-to-understand summaries.

### 📝 Quiz Generator
Generate multiple-choice quizzes on any topic for self-assessment.

### 🧠 Flashcards
Create AI-generated flashcards for quick revision.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Shahz-ft/SkillIT-AI.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **backend** folder.

```env
OPENROUTER_API_KEY=YOUR_API_KEY
```
